# Math operations

Module to do basic math operations

## Installation

```
npm install project-1a-math

```

## Use

Follow this instructions

```
//module import
const m = require(".");

// addition
console.log(m.add(25, 26)); =>

// subtraction
console.log(m.sub(2020, 1995)); => 25

// multiplication
console.log(m.mul(10, 15)); => 150

// division
console.log(m.div(933, 3)); => 311

```
